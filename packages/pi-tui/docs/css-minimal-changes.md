# CSS Changes for Minimal Style

## Current Issues
- Heavy borders create visual clutter
- All blocks have same spacing - no hierarchy
- Tool blocks not visually distinct from messages

## Target Style (Claude Code inspired)
- Subtle borders or no borders
- More whitespace between message blocks
- Tool blocks slightly indented
- Reduced color emphasis

## Changes
1. Remove heavy borders, use subtle ones
2. Increase message block margin
3. Add left padding to tool blocks
4. Reduce color saturation
